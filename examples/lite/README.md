# Lite Example

This example shows how to use Kabeeri VDF for a small MVP or simple tool.

The original `examples/lite/` folder was a placeholder. This improved version gives beginners a clear first example.

## Example product

```text
Simple Booking Request App
```

## Product idea

A small business wants a simple web app where visitors can submit a booking request.

The business owner can later review requests manually.

## Why Lite fits

Use Lite because the first version does not need:

```text
- complex roles
- advanced admin dashboard
- multi-tenant architecture
- billing
- complex reporting
- multiple integrations
```

## First release scope

```text
- Landing page
- Booking request form
- Basic confirmation message
- Simple admin/review workflow or email notification
- Basic acceptance checklist
```

## Out of scope

```text
- Online payments
- Customer accounts
- Calendar sync
- Advanced notifications
- Multi-branch management
- Analytics dashboard
```

## Suggested Kabeeri VDF flow

```text
1. Choose Lite generator.
2. Answer simple product questions.
3. Create only essential planning documents.
4. Choose one prompt pack.
5. Create 2 to 4 implementation tasks.
6. Review AI output with acceptance checklist.
7. Commit and close the issue.
```
