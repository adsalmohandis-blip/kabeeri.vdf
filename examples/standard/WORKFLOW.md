# Standard Example Workflow

## Step 1 — Choose profile

```text
standard
```

## Step 2 — Choose stack

Example stack:

```text
Laravel backend
Next.js frontend
```

Alternative:

```text
FastAPI backend
React frontend
```

## Step 3 — Create planning documents

Use Kabeeri VDF documents for:

```text
Strategy
Architecture
Release specs
Database architecture
Execution plan
Engineering standards
AI code prompts
Task tracking
Acceptance checklists
```

## Step 4 — Create tasks

```text
STD-T001 - Create appointment request database model
STD-T002 - Create appointment request API
STD-T003 - Create patient request form
STD-T004 - Create staff login
STD-T005 - Create staff dashboard
STD-T006 - Review and release handoff
```

## Step 5 — Use prompt packs

If using Laravel + Next.js:

```text
prompt_packs/laravel/
prompt_packs/nextjs/
```

## Step 6 — Track and accept

Use:

```text
task_tracking/TASK_TEMPLATE.md
acceptance_checklists/TASK_COMPLETION_ACCEPTANCE_CHECKLIST.md
acceptance_checklists/AI_OUTPUT_ACCEPTANCE_CHECKLIST.md
```
