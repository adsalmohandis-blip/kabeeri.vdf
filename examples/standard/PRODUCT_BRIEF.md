# Standard Example Product Brief

## Product name

```text
Clinic Booking Portal
```

## Product goal

Help patients request appointments online and help staff manage those requests.

## Target users

```text
- Patients
- Clinic staff
- Clinic manager
```

## First version features

```text
- Appointment request form
- Staff login
- Appointment request dashboard
- Status update
- Staff notes
- Basic email or manual follow-up
```

## Main records

```text
AppointmentRequest
StaffUser
Service
StatusHistory
```

## Suggested statuses

```text
new
contacted
confirmed
cancelled
```

## Not included in V1

```text
- Payments
- Insurance
- Patient records
- Prescription management
- SMS gateway
- Multi-location management
```
