# Task Definition of Ready

A task is ready only when all checks pass.

## Checklist

- [ ] Task has one clear outcome.
- [ ] Task has a valid source reference.
- [ ] Task has source mode: direct, indirect, derived, suggested, or required.
- [ ] Scope includes included and excluded work.
- [ ] Allowed files and forbidden files are listed.
- [ ] Acceptance criteria are testable.
- [ ] Assignee is known.
- [ ] Reviewer is not the assignee.
- [ ] Delivery mode and intake mode are recorded.
- [ ] AI usage budget is present when AI work is expected.

Tasks that fail this checklist stay in proposed or blocked state.
