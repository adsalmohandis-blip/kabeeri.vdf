# Final Validation Checklist

## No Summary-Only Execution
Codex must use roadmap source documents and Requirements Traceability Matrix before implementation.

## No Silent Overwrite
No existing file is replaced without reason, backup note, and report.

## No Raw Design Execution
PDF/images/Drive links/reference websites must become approved text specs before frontend implementation.

## Owner Verify
Final task verification, release approval, publish approval, and owner transfer require Owner rules.

## Cost Awareness
AI usage must be traceable by task, sprint, workstream, developer/agent, provider/model, and random/untracked usage.

## Vibe First
CLI must be optional/hidden behind UI/VS Code/Dashboard/Chat flows for normal users.

## Docs Ready
User documentation must explain how to start, continue, monitor, verify, control cost, and publish.

## GitHub Safe
GitHub mutation only after auth + owner approval; otherwise generate import-ready files.
