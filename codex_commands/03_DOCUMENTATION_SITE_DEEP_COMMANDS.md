# Kabeeri-vdf Documentation Site Deep Command

Build a static HTML/CSS/JavaScript documentation site.

Required:
- Arabic and English
- language switcher in navbar
- sidebar navigation
- RTL support
- no backend
- local-first
- pages covering: start here, new project, existing project, Structured, Agile, questionnaires, tasks, dashboard, owner verify, cost control, multi-AI, vibe UX, design source governance, production vs publish, troubleshooting.

Codex command:

```text
Build a static documentation site for Kabeeri-vdf.
No backend.
Use HTML/CSS/JavaScript only.
Create docs_site/index.html, docs_site/assets/css/style.css, docs_site/assets/js/app.js, and pages in Arabic and English.
Make the site useful for a new user who needs to know where to start, how to continue, how to monitor work, how to verify tasks, and how to publish.
```
