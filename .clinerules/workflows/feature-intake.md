# Feature Intake

1. Read the active Evolution priority and current Kabeeri development state.
2. Check whether a priority is already in progress.
3. Show the current priority list and point out the active item.
4. Recommend a placement for the new feature request.
5. Wait for explicit Owner confirmation before implementing or reordering anything.
6. After confirmation, record the feature in Evolution with a durable task description.
7. Keep deferred ideas separate unless the Owner restores them into the active priorities.
