# Commercial Boundary

Kabeeri VDF is the open-source AI development governance core.

KVDOS is a separate commercial product built on top of Kabeeri VDF concepts and may consume VDF through a package dependency, CLI bridge, or API bridge.

## Included in Kabeeri VDF

- CLI governance engine
- Basic local runtime state under `.kabeeri/`
- Task governance and basic task lifecycle
- Prompt pack format
- Questionnaires and project intake guidance
- Project profiles
- Schemas and output contracts
- Basic validation and reports
- Open-source developer documentation

## Not Included in Kabeeri VDF

- KVDOS Studio
- KVDOS Runner
- KVDOS Cloud
- Billing and subscriptions
- Commercial licensing
- Marketplace
- Signed package registry
- Enterprise self-hosting
- Secure commercial runtime
- Advanced commercial agents
- Private knowledge base

## Rule

A feature belongs in Kabeeri VDF when it improves the open governance core for AI-assisted development.

A feature belongs in KVDOS when it depends on commercial distribution, licensing, billing, cloud services, enterprise deployment, proprietary product UX, or private commercial runtime behavior.
