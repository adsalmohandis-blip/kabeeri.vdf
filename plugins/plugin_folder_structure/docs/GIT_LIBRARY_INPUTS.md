# Git Library Inputs

Git library inputs are governed records, not blind source copies.

Owner Track keeps them under `docs/git_libraries/`.

Plugin Development Track keeps them under `inputs/git_libraries/`.

The workspace keeps:

- `inputs/git_libraries/libraries.md`
- `inputs/git_libraries/selected_libraries.json`
- `inputs/git_libraries/adoption_decisions.md`
- review folders for analysis, license, security, compatibility, and maintenance
