# AI Learning Plugin

This plugin keeps reusable AI learning close to KVDF Core without becoming the source of truth for planning.

It is designed for:

- local learning capture
- sanitized export packages and Markdown summaries
- owner review, promotion, and rejection
- shared learning and local cache support
- future cloud-provider metadata

The canonical memory file still lives in `.kabeeri/ai_learning/failure_patterns.json`.
Shared KVDF knowledge lives in `knowledge/ai_learning/` and only changes after Owner approval.
