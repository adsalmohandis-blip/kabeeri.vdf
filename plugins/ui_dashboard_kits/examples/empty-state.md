# Empty State Recipe

Use when a data surface has zero records or no results.

## Required Elements

1. State icon.
2. Clear title.
3. Short explanation.
4. Primary action when the user can create or recover.
5. Secondary action when useful, such as Clear filters or Learn more.

## Acceptance Criteria

- Empty state is not just plain text.
- Icon is muted and accessible or decorative with hidden text.
- Copy explains why the state exists.
- Primary action uses icon and semantic button style.
- Search no-results state offers filter reset.

## Variation Hooks

Vary tone from product context:

- operational: direct and concise;
- ecommerce: conversion-friendly;
- enterprise: formal and clear;
- clinical: calm and reassuring;
- playful product: light but not silly.
