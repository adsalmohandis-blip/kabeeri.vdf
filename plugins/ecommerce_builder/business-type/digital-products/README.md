# Digital Products

Ecommerce flows for downloadable or licensed digital products.

Intake questions:
- Is the product a download, license, template, or access pass?
- Do you need activation, license keys, or expiry rules?
- Should customers access files from their account page?
- Do you need upsells, bundles, or version downloads?

Module map:
- product catalog
- product detail and preview
- checkout and payment
- license or download fulfillment
- account library
- admin fulfillment settings

Sample acceptance criteria:
- Digital product purchase can be completed end to end.
- Access or download delivery is available after payment.
- License or expiry rules are enforced.
- Admin can manage files, keys, or access rules.

Starter task pack:
- define product delivery and access rules
- build checkout and fulfillment flow
- add account library and download access
- create admin access controls
