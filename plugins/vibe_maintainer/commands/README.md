# Vibe Maintainer Commands

## `cleanup`

Run the workspace maintenance audit and save the report.

## `inspect`

Run a file-by-file inspection without the full audit summary.

## `report`

Show the saved summary from the latest maintenance pass.

## `relocate`

Preview or apply safe folder relocation candidates.

## `approve`, `execute`, `finalize`

Move the maintenance pass through the approval and completion lifecycle.
