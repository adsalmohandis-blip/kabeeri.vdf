# Cafe

Cafe POS packs are for fast order entry, pickup queues, and combo items.

Intake questions:
- Do you need queue or pickup workflows?
- Should combo items and modifiers be included?
- How fast must the register flow be?

Module map:
- quick order register
- pickup queue
- combo item catalog
- payment and receipt
- shift close

Sample acceptance criteria:
- Orders can be entered quickly.
- Pickup queues are visible.
- Combo or modifier behavior is supported.

Starter task pack:
- define cafe ordering and pickup flow
- build quick register and queue
- add combos and receipt handling
