# Retail

Retail POS packs are for barcode checkout, stock control, and returns.

Intake questions:
- Do you need barcode scanning and stock counts?
- Should returns and exchanges be supported?
- What payment and receipt rules matter?

Module map:
- register checkout
- item lookup
- inventory sync
- returns and exchanges
- shift closing

Sample acceptance criteria:
- Items can be scanned or searched quickly.
- Returns are handled safely.
- Shift close produces a usable summary.

Starter task pack:
- define retail checkout and stock rules
- build register and item lookup flow
- add returns and shift closing
