# Restaurant

Restaurant POS packs are for table service, kitchen routing, and split bills.

Intake questions:
- Do you need table management or floor plans?
- Should orders route to the kitchen or bar?
- Do you need split bills or tipping rules?

Module map:
- table map
- order entry
- kitchen routing
- split bill and payments
- shift reporting

Sample acceptance criteria:
- Tables and orders are easy to manage.
- Kitchen routing is visible.
- Split bill behavior works correctly.

Starter task pack:
- define table and routing rules
- build order and kitchen screens
- add bill split and closing flow
