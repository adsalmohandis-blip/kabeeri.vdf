# Workflow

1. Check that the workspace exists and matches the `plugin_folder_structure` contract.
2. Collect intake.
3. Analyze libraries.
4. Generate specs.
5. Generate governed tasks.
6. Build source one task at a time.
7. Validate contracts and integrations.
8. Run tests.
9. Collect evidence.
10. Score readiness.
11. Build a package.
12. Request Owner Track promotion.
