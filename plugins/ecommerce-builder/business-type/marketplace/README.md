# Marketplace

Multi-vendor ecommerce flows with vendor, commission, and marketplace management.

Intake questions:
- How do vendors onboard and manage listings?
- Do you need commissions, payouts, or marketplace fees?
- Should buyers compare vendors on one listing page?
- Do you need moderation for products, vendors, or reviews?

Module map:
- marketplace home
- vendor storefronts
- listing and catalog aggregation
- commission and payout rules
- checkout and order routing
- marketplace admin

Sample acceptance criteria:
- Buyers can browse by vendor and marketplace category.
- Vendor listings are isolated and manageable.
- Orders route to the correct vendor or fulfillment path.
- Admin can manage payouts, moderation, and commissions.

Starter task pack:
- define vendor onboarding and catalog rules
- build marketplace browse and listing flow
- add order routing and commission logic
- create admin moderation and payout controls
