# Ecommerce Types

This folder stores the ecommerce archetype playbooks that shape questionnaire, brief, design, modules, tasks, and acceptance.

Use it for:

- store
- marketplace
- digital products
- subscription
- services

Shared intake themes:
- what is being sold
- how orders are fulfilled
- what payment and tax rules apply
- what customer account behavior is required
- what inventory, access, or subscription model is used

Shared module themes:
- catalog
- product detail
- cart
- checkout
- orders
- account
- admin

Shared acceptance themes:
- product discovery works
- checkout is consistent and secure
- order or access state is recorded
- admin can manage catalog or fulfillment
- edge cases are handled clearly
