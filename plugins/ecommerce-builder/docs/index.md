# Ecommerce Builder

`ecommerce-builder` is the first app-track plugin for building ecommerce systems with a CLI-first, plugin-first pipeline.

## Intended scope

- Storefronts
- Single-vendor ecommerce
- Digital products stores
- Subscription commerce
- Service commerce
- Marketplace commerce

## Core promise

Turn one vibe request into:

1. a questionnaire
2. a product brief
3. a UI/UX brief
4. a system design brief
5. a module breakdown
6. proposed tasks
7. approval-ready task batches

## Lifecycle

- install the plugin
- enable the plugin
- build an ecommerce project
- disable or uninstall the plugin when no longer needed

The live runtime state is stored in `.kabeeri/ecommerce.json`.
