# Third-Party Notices

This plugin provides optional GitHub remote-provider guidance and planning for KVDF.

It does not bundle GitHub source code.
It does not make GitHub mandatory for KVDF Core.
It keeps GitHub-specific provider behavior isolated in an optional removable plugin.

GitHub platform/API usage may rely on the user's local `gh` CLI or GitHub
credentials when the operator explicitly confirms remote actions.

No additional third-party assets are bundled in this plugin.
