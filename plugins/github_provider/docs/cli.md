# GitHub Provider CLI

## Status

```bash
kvdf github-provider status --json
```

## Readiness

```bash
kvdf github-provider readiness --json
```

## Sync Plan

```bash
kvdf github-provider sync-plan --json
```

## Issue Plan

```bash
kvdf github-provider issue-plan --json
```

## PR Plan

```bash
kvdf github-provider pr-plan --json
```

## Release Plan

```bash
kvdf github-provider release-plan --json
```

## Handoff Plan

```bash
kvdf github-provider handoff-plan --json
```

## Compatibility

```bash
kvdf github --json
```

The legacy `kvdf github` command is preserved as a wrapper around the
canonical `github_provider` plugin.
