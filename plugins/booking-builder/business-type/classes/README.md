# Classes

Booking flows for classes, workshops, cohorts, and scheduled learning sessions.

Intake questions:
- Is this one-off classes or multi-session cohorts?
- Do you need waitlists, capacity limits, or prerequisites?
- Should learners register individually or as a group?
- Do you need certificates, attendance, or post-class follow-up?

Module map:
- class catalog
- schedule and capacity
- registration flow
- cohort management
- attendance tracking
- admin scheduling

Sample acceptance criteria:
- Users can view class dates and remaining capacity.
- Registration respects capacity and waitlist rules.
- Admin can manage cohorts and sessions.
- Attendance or completion can be tracked after booking.

Starter task pack:
- define class schedule and capacity rules
- build registration and waitlist flow
- add admin cohort management
- wire attendance or completion tracking
