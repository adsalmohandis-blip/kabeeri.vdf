# Events

Booking flows for events, tours, classes, and attendance-based reservations.

Intake questions:
- Is this ticketed entry, RSVP, or scheduled tour booking?
- Do you need capacity controls and ticket types?
- Should event pages support add-ons or group booking?
- Do you need attendee check-in or post-event follow-up?

Module map:
- event catalog
- event detail
- ticket or RSVP flow
- capacity and seat rules
- confirmation and ticketing
- admin event management

Sample acceptance criteria:
- Users can book or RSVP for an event.
- Capacity is enforced before confirmation.
- Ticket or confirmation output is generated.
- Admin can edit event details and ticket settings.

Starter task pack:
- define event types and capacity rules
- build event detail and registration flow
- add ticketing or RSVP confirmation
- create admin tools for event updates
