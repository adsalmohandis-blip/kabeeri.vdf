# Appointments

Booking flows for appointment-based products such as clinics, salons, consultants, and service providers.

Intake questions:
- Do users book with staff, locations, or both?
- Are there fixed slots, flexible durations, or recurring visits?
- Do you need deposits, reminders, or cancellation rules?
- Should bookings support intake forms before confirmation?

Module map:
- service catalog
- staff or provider profiles
- calendar and slot picker
- booking form and confirmation
- reminder and reschedule flow
- admin schedule management

Sample acceptance criteria:
- A customer can select a provider and time slot.
- The system blocks double booking.
- Confirmation and reminder behavior is visible.
- Admin can move or cancel appointments safely.

Starter task pack:
- define appointment types and duration rules
- design provider availability and slot inventory
- build booking and confirmation flow
- add admin calendar controls
