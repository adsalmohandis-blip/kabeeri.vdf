# Third-Party Notices

This plugin is built from scratch for KVDF Core.

- `ui_ux_intelligence` is a standalone KVDF plugin.
- No external GitHub dependency is required for the relocated catalog or search engine.
- Only the selected CSV data files are included in `data/` and `data/stacks/`.
- Reference scripts and templates were used only as implementation guidance and were not copied into runtime code.
- Font binaries, screenshots, previews, packaging, and unrelated wrappers were excluded.
