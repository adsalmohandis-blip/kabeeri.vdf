# Company Profile Builder

`company-profile` is the app-track plugin for company profile websites.

The plugin-owned runtime starts from:

- [plugins/company-profile/bootstrap.js](/D:/My%20Project%20Ideas/kabeeri.vdf/kabeeri-vdf/plugins/company-profile/bootstrap.js)
- [plugins/company-profile/runtime/company_profile.js](/D:/My%20Project%20Ideas/kabeeri.vdf/kabeeri-vdf/plugins/company-profile/runtime/company_profile.js)

```bash
kvdf plugins install company-profile
kvdf plugins uninstall company-profile
kvdf company-profile status
kvdf company-profile init --mode corporate
```

The live runtime state lives in `.kabeeri/company_profile.json`.
