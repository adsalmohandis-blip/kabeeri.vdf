# Franchise

Franchise profiles are for businesses that need local branch pages, franchise lead capture, and territory-aware navigation.

Intake questions:
- Do you need location pages for branches?
- Should the site support franchise recruitment or partner applications?
- What local and brand-level content must be separated?

Module map:
- brand overview
- franchise opportunity page
- location pages
- local inquiry flow
- branch or territory admin

Sample acceptance criteria:
- Users can discover branch locations easily.
- Franchise inquiry forms route to the right team.
- Brand and local content are clearly separated.

Starter task pack:
- define locations and franchise path
- build location pages and inquiry forms
- add branch-level content controls
