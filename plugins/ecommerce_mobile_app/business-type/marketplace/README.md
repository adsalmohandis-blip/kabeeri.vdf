# Marketplace

Marketplace apps connect buyers to multiple vendors and collections.

Intake questions:
- Do you need vendor profiles or seller pages?
- Should collections or curated sets be promoted?
- How should orders route between sellers?

Module map:
- vendor browsing
- marketplace catalog
- product detail
- cart and checkout
- vendor or collection pages

Sample acceptance criteria:
- Users can browse products by vendor or collection.
- Vendor identity is visible.
- Checkout can route marketplace orders correctly.

Starter task pack:
- define vendor and collection structure
- build marketplace browse and detail screens
- add routing rules for vendor orders
