# Loyalty

Loyalty apps focus on rewards, points, offers, and retention.

Intake questions:
- How are points earned and redeemed?
- Do you need referrals, coupons, or offers?
- Should push campaigns or wallet-style access be included?

Module map:
- points balance
- rewards catalog
- redemption flow
- offers and coupons
- account and activity log

Sample acceptance criteria:
- Users can view points and rewards clearly.
- Redemption rules are understandable.
- Offers can be surfaced through the app.

Starter task pack:
- define points and redemption rules
- build rewards and offers screens
- add activity history and offer hooks
