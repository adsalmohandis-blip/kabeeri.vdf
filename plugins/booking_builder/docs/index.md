# Booking Builder

`booking-builder` is the first app-track plugin for building booking websites with a CLI-first, plugin-first pipeline.

## Intended scope

- appointments
- services booking
- classes and workshops
- hotel and room booking
- tours and events

## Core promise

Turn one vibe request into:

1. a questionnaire
2. a booking brief
3. a UI/UX brief
4. a system design brief
5. a module breakdown
6. proposed tasks
7. approval-ready task batches

## Lifecycle

- install the plugin
- enable the plugin
- build a booking project
- disable or uninstall the plugin when no longer needed
