# Newsletter

Newsletter-led sites focus on subscriber capture, issue archives, and email-first publishing.

Intake questions:
- Is the main product a newsletter or a broader news site?
- Do you need issue archives and sign-up funnels?
- What segments or subscription tiers matter?

Module map:
- subscriber signup
- issue archive
- issue detail page
- editorial send workflow
- subscriber admin

Sample acceptance criteria:
- Visitors can subscribe from the homepage.
- Past issues are easy to browse.
- Editors can prepare and manage newsletter sends.

Starter task pack:
- define subscription and archive flow
- build sign-up and issue pages
- add send workflow and subscriber admin
