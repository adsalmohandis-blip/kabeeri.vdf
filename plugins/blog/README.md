# Blog Builder

`blog` is the app-track plugin for blogs and editorial publishing.

The plugin-owned runtime starts from:

- [plugins/blog/bootstrap.js](/D:/My%20Project%20Ideas/kabeeri.vdf/kabeeri-vdf/plugins/blog/bootstrap.js)
- [plugins/blog/runtime/blog.js](/D:/My%20Project%20Ideas/kabeeri.vdf/kabeeri-vdf/plugins/blog/runtime/blog.js)

```bash
kvdf plugins install blog
kvdf plugins uninstall blog
kvdf blog status
kvdf blog init --mode personal
```

The live runtime state lives in `.kabeeri/blog.json`.
