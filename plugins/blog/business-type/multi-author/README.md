# Multi-Author

Multi-author blogs are for teams that need editorial review, author roles, and shared publishing.

Intake questions:
- How many authors will publish content?
- Do you need editorial review before publishing?
- Should author pages and contributor profiles be included?

Module map:
- author profiles
- draft and review workflow
- post publishing
- editorial queue
- admin roles

Sample acceptance criteria:
- Multiple authors can work independently.
- Editors can review drafts before publication.
- Author attribution is clear on every post.

Starter task pack:
- define author roles and review steps
- build contributor and editorial pages
- add draft review and publish flow
