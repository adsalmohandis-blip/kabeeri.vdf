# Personal

Personal blogs center the writer's voice, reading comfort, and simple publishing.

Intake questions:
- What topics or personal themes should the blog cover?
- Do you need comments or newsletter signup?
- Should the tone feel casual, reflective, or expert?

Module map:
- home feed
- post page
- archive and tags
- about the author
- simple admin editor

Sample acceptance criteria:
- Posts are easy to read on desktop and mobile.
- The author identity is visible.
- Readers can browse archives or tags.

Starter task pack:
- define the writing lane and archive structure
- build the home, post, and about pages
- add lightweight newsletter or comments
