# Vibe Maintainer Docs

This plugin keeps vibe developer app workspaces healthy.

## Commands

- `cleanup`
- `inspect`
- `report`
- `relocate`
- `approve`
- `execute`
- `finalize`

## Scope

Choose one of:

- `--workspace <slug>`
- `--workspaces <slug-a,slug-b>`
- `--all`

## Modes

- `fast` = quick scan
- `slow` = strict scan with deeper review
