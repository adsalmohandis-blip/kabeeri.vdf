# Shopping

Shopping apps focus on fast catalog browsing, product discovery, and quick checkout.

Intake questions:
- What source commerce system should the app use?
- Do you need guest checkout or saved carts?
- Should browsing or checkout be the main priority?

Module map:
- catalog browsing
- product detail
- cart and checkout
- account and orders
- push notifications

Sample acceptance criteria:
- Users can browse products and checkout quickly.
- Saved cart or wishlist behavior is visible.
- Account history is easy to reach.

Starter task pack:
- define shopping flow and cart rules
- build catalog and checkout screens
- add order history and notification hooks
