# Ecommerce Mobile App Builder

`ecommerce-mobile-app` is the app-track plugin for mobile commerce apps.

```bash
kvdf plugins install ecommerce-mobile-app
kvdf plugins uninstall ecommerce-mobile-app
kvdf ecommerce-mobile-app status
kvdf ecommerce-mobile-app init --mode shopping
```

The live runtime state lives in `.kabeeri/ecommerce_mobile_app.json`.
