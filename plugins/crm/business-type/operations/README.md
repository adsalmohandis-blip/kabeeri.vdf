# Operations

Operations CRM packs are for internal requests, approvals, and service queues.

Intake questions:
- What internal processes need tracking?
- Do you need approval queues or service levels?
- Which teams own each operational step?

Module map:
- request intake
- approval queue
- service tracking
- operational dashboard
- audit trail

Sample acceptance criteria:
- Requests can move through an approval process.
- Status is visible to the owning team.
- Audit history is retained.

Starter task pack:
- map request and approval stages
- build queue and dashboard views
- add audit trail and status rules
