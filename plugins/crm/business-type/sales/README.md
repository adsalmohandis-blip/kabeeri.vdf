# Sales

Sales CRM packs are for lead management, pipeline tracking, and forecasting.

Intake questions:
- What lead sources and pipeline stages matter?
- Do you need forecasting or owner assignment?
- Should automation score or route leads?

Module map:
- leads and contacts
- pipeline stages
- forecast views
- task follow-up
- sales reports

Sample acceptance criteria:
- Leads can move through pipeline stages.
- Forecast views summarize open deals.
- Follow-up tasks are visible to owners.

Starter task pack:
- define pipeline stages and lead sources
- build pipeline and forecast views
- add task and reminder flows
