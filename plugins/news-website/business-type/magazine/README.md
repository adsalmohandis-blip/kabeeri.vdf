# Magazine

Magazine sites focus on long-form features, strong visuals, and themed storytelling.

Intake questions:
- Do you need issue-based or themed editions?
- What image-heavy or feature-led layout style should be used?
- Which monetization paths matter most?

Module map:
- featured home layout
- issue or edition pages
- feature story pages
- author and series pages
- subscriber or newsletter signup

Sample acceptance criteria:
- Feature content is visually emphasized.
- The site supports themed or issue-based presentation.
- Readers can subscribe or sign up from key pages.

Starter task pack:
- define theme and issue structure
- build feature-led home and story pages
- add issue archive and signup flow
