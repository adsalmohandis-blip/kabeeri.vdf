## Tailwind CSS

- Name: Tailwind CSS

- License: MIT

Tailwind CSS is now treated as an optional, removable `tailwind_ui` plugin provider for guidance, snippets, and future generated HTML surfaces.

KVDF Core no longer has a hard Tailwind dependency. Tailwind is not bundled as a Core package dependency or Core devDependency.
