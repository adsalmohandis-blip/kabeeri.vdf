# Flutter Prompt Pack

This directory contains the first Flutter prompt pack for **Kabeeri Vibe Developer Framework**.

The pack is designed for vibe developers and AI-powered builders who want to build mobile apps using Flutter after completing Kabeeri VDF planning documents.

## Important clarification

This prompt pack is **not** a Flutter installer.

It does not install Flutter SDK, Dart, Android Studio, Xcode, dependencies, or create a Flutter project automatically.

It provides structured AI prompts that help an AI coding assistant work on an existing or separately prepared Flutter project.

## Purpose

This prompt pack helps the user move from planning to mobile implementation in small, controlled AI coding tasks.

Each prompt should be used one at a time.

## Core rule

Do not ask an AI coding tool to build the whole Flutter app at once.

Use this flow:

```text
One prompt
→ one small mobile task
→ review output
→ run checks
→ test in emulator/simulator/device
→ commit
→ move to next prompt
```
