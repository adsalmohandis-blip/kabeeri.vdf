# Vue.js Prompt Pack

This directory contains the first Vue.js prompt pack for **Kabeeri Vibe Developer Framework**.

This pack is for projects using Vue as a frontend SPA or UI layer without requiring Next.js.

## Important clarification

This prompt pack is **not** a Vue installer.

It does not install Node.js, create a Vite/CRA project, install packages, configure hosting, or deploy automatically.

It provides structured AI prompts that help an AI coding assistant work on an existing or separately prepared Vue project.
