# Angular Prompt Pack

This directory contains the first Angular prompt pack for **Kabeeri Vibe Developer Framework**.

This pack is for Angular frontend applications, enterprise dashboards, admin panels, and frontend layers connected to APIs.

## Important clarification

This prompt pack is **not** an Angular installer.

It does not install Node.js, Angular CLI, create a project, install packages, configure hosting, or deploy automatically.

It provides structured AI prompts that help an AI coding assistant work on an existing or separately prepared Angular project.
