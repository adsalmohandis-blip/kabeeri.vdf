# Express.js Prompt Pack

This directory contains the first Express.js prompt pack for **Kabeeri Vibe Developer Framework**.

The pack is designed for vibe developers and AI-powered builders who want to use Express.js as a Node.js backend/API implementation stack after completing Kabeeri VDF planning documents.

## Important clarification

This prompt pack is **not** an Express.js installer.

It does not install Node.js, install packages, create an Express project, configure hosting, or deploy anything automatically.

It provides structured AI prompts that help an AI coding assistant work on an existing or separately prepared Express.js project.

## What Express.js can support

This pack can help with:

- REST API structure
- routing
- middleware
- controllers
- services
- database integration
- authentication
- roles and permissions
- validation
- error handling
- logging
- integrations
- testing and release review

## Core rule

Do not ask an AI coding tool to build the whole Express.js backend at once.

Use this flow:

```text
One prompt
→ one small backend task
→ review output
→ run checks/tests
→ commit
→ move to next prompt
```
