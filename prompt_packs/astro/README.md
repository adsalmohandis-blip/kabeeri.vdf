# Astro Prompt Pack

This directory contains the first Astro prompt pack for **Kabeeri Vibe Developer Framework**.

The pack is designed for vibe developers and AI-powered builders who want to use Astro for content websites, landing pages, documentation sites, marketing sites, blogs, and hybrid frontend projects after completing Kabeeri VDF planning documents.

## Important clarification

This prompt pack is **not** an Astro installer.

It does not install Node.js, create an Astro project, run package manager commands, configure hosting, or deploy anything automatically.

It provides structured AI prompts that help an AI coding assistant work on an existing or separately prepared Astro project.

## What Astro can support

This pack can help with:

- marketing websites
- landing pages
- documentation sites
- blogs
- content collections
- static pages
- hybrid islands
- CMS integration
- SEO and metadata
- forms
- deployment handoff
- testing and review

## Core rule

Do not ask an AI coding tool to build the whole Astro site at once.

Use this flow:

```text
One prompt
→ one small task
→ review output
→ run checks
→ test locally
→ commit
→ move to next prompt
```
