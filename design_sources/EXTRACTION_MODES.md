# Extraction Modes

## Manual Spec Mode

A human reads the design source and writes the text spec manually. This is safest for sensitive, client-facing, or high-brand-precision work.

## Assisted Spec Mode

An AI vision or chat tool drafts a spec from the source. A human must review, correct, and approve the draft before implementation.

## Automated Spec Mode

Future adapters may extract draft specs automatically. Automated output is still a draft until approved by a human.

## Rule

No extraction mode authorizes direct frontend implementation from raw visual sources.
