# Do Not Copy Rules

AI agents must not copy protected or client-owned design work unless the project has explicit rights and an approved spec.

## Do Not Copy

- third-party website layouts
- brand colors without rights
- proprietary illustrations
- product screenshots
- exact copywriting
- paid template designs
- icons or assets without license

## Safe Alternative

Extract intent, structure, and usability goals, then create an original approved text spec using the project design system.
