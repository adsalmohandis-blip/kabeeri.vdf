# Reference Website Inspiration Rules

Reference websites are inspiration sources only.

## Allowed

- describe interaction patterns
- note information hierarchy
- capture general mood words
- compare density or navigation style
- identify accessibility or responsiveness ideas

## Forbidden

- copying layout exactly
- copying brand identity
- copying images, icons, or text
- recreating proprietary components
- treating the reference URL as an implementation spec

## Required Output

Reference website notes must become original text specs and must state what is inspired, what is excluded, and what must be different.
