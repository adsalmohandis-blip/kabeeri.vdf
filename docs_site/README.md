# Kabeeri VDF Documentation Site

This is a static local documentation site. It has no backend and no build step.

## Open Locally

Open `docs_site/index.html` in a browser.

## Structure

- `index.html`
- `assets/css/style.css`
- `assets/js/app.js`
- `pages/en/*.html`
- `pages/ar/*.html`

## Coverage

The site covers start workflows, delivery modes, questionnaire engine, task governance, dashboard monitoring, Owner verify, AI cost control, Multi-AI governance, Vibe UX, design source governance, production versus publish, and troubleshooting.
