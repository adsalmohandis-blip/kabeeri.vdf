# Design Source Snapshot Rules

Snapshots preserve what was approved.

## Required Snapshot Metadata

- source ID
- source type
- captured by
- captured at
- original location
- version or export reference
- file names or links
- approval actor
- approval date
- checksum when available

## External Sources

For Figma, Drive, Framer, Webflow, or websites, record the exact version, export, frame, page URL, or screenshot set used for approval.

## Rule

If the source changes, create a new snapshot. Do not overwrite the old approval trail.
