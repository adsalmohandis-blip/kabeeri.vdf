# UI Decision Review Checklist

- High-priority UI decision questions have answers or are intentionally deferred.
- Decision profile selected a creative variant from approved archetypes.
- Palette, density, navigation, surface style, and tone match answers.
- The selected variant still uses approved tokens, composition, and adapter.
- Arabic/RTL answers enable RTL Arabic UI references.
- High-risk answers reduce motion and strengthen status/error clarity.
- Missing answers are listed as questions, not guessed silently.
