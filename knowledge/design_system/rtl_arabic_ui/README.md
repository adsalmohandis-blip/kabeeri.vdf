# RTL Arabic UI Reference Pack

Use this pack when a product supports Arabic, RTL, bilingual UI, MENA audiences, Arabic dashboards, Arabic commerce, Arabic forms, or Arabic content-heavy surfaces.

The goal is not only to flip layout direction. Arabic UI needs typography, spacing, icon behavior, number/date rules, table behavior, form ergonomics, and accessibility decisions.

Core order:

1. Decide if the screen is Arabic-only, English-only, or bilingual.
2. Apply `RTL_ARABIC_UI_CONTRACT.md`.
3. Apply typography and layout rules.
4. Review forms, tables, navigation, icons, motion, and accessibility.
5. Run `RTL_REVIEW_CHECKLIST.md`.

Files:

- `RTL_ARABIC_UI_CONTRACT.md`
- `ARABIC_TYPOGRAPHY.md`
- `RTL_LAYOUT_PATTERNS.md`
- `RTL_COMPONENT_RULES.md`
- `ARABIC_FORMS_TABLES.md`
- `RTL_ACCESSIBILITY.md`
- `RTL_REVIEW_CHECKLIST.md`
