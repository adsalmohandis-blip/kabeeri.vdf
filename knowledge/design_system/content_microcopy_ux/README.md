# Content And Microcopy UX Pack

Use this pack for every UI implementation, refactor, design recommendation, or UI review.

The goal is to make interface text useful, consistent, and product-aware. Strong UI copy reduces support load, improves conversion, prevents user mistakes, and makes generated designs feel intentionally made for the product instead of generic.

Core order:

1. Choose the microcopy level and tone for the product.
2. Apply `CONTENT_MICROCOPY_CONTRACT.md`.
3. Apply action labels, empty states, errors, forms, onboarding, confirmations, and status messaging rules.
4. Align copy with business risk, language, and user intent.
5. Run `CONTENT_REVIEW_CHECKLIST.md` before delivery.

Files:

- `CONTENT_MICROCOPY_CONTRACT.md`
- `ACTION_LABEL_RULES.md`
- `EMPTY_ERROR_STATE_COPY.md`
- `FORM_VALIDATION_COPY.md`
- `ONBOARDING_HELP_COPY.md`
- `CONFIRMATION_STATUS_COPY.md`
- `BUSINESS_TONE_MATRIX.md`
- `CONTENT_REVIEW_CHECKLIST.md`
