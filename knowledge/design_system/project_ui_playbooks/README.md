# Project UI Playbooks

This pack maps every Kabeeri product blueprint to a compact UI/UX playbook.

Use it before page implementation to avoid re-deciding the same design direction for common project types.

## Commands

```bash
kvdf design playbooks
kvdf design playbook erp --json
kvdf design playbook ecommerce --page checkout --json
```

## Rule

The playbook is a default starting point, not a rigid template. UI Decision Intake and developer answers can override variant, density, navigation, palette, and tone.
