# Kabeeri UI Flows

UI flows capture the user journey, not just the screen shape. Use them with business patterns and UI recipes.

Core order:

1. Pick the business pattern.
2. Pick the primary flow.
3. Pick view recipes.
4. Add required states and confirmations.
5. Run UI and motion review.

Available flows:

- `AUTH_FLOW.md`
- `CRUD_FLOW.md`
- `DELETE_CONFIRMATION_FLOW.md`
- `USER_MANAGEMENT_FLOW.md`
- `ONBOARDING_FLOW.md`
- `CHECKOUT_FLOW.md`
- `PAYMENT_CONFIRMATION_FLOW.md`
- `BOOKING_FLOW.md`
- `AI_PRODUCT_FLOW.md`
