# Creative Variant Intelligence

This pack generates bounded creative directions for a UI screen or product.

It prevents repetitive AI-generated UI while keeping implementation governed by:

- Theme Token Intelligence;
- Component Composition Intelligence;
- Framework Adapter Intelligence;
- UI Execution Kit;
- accessibility, performance, content, motion, and RTL rules.

## Commands

```bash
kvdf design variant-archetypes
kvdf design variants ecommerce --page checkout --count 3 --json
kvdf design variants erp --page "invoice approval table" --framework antd
```

## Rule

Variants are design directions, not random redesigns. Each variant must change at least four creative axes while preserving the same required states, accessibility, framework compatibility, and business workflow.
