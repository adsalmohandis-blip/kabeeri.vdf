# Marketplace UI Pattern Pack

Use this pack for multi-vendor commerce, service marketplaces, listing platforms, and buyer/seller products.

## Design Posture

- Priority: discovery, seller trust, comparison, transaction clarity.
- Density: balanced.
- Motion: `BALANCED_MOTION`.
- Dashboard reference: `ADMIT-ADB02` for marketplace admin and analytics.

## Required Views

- Listings
- Listing detail
- Seller profile
- Buyer request/checkout
- Reviews and dispute/support

## Required Flows

- `CHECKOUT_FLOW.md`
- `USER_MANAGEMENT_FLOW.md`

## Creative Axes

- Trust signal placement
- Listing comparison style
- Seller identity prominence
- Filter depth
- Review and dispute visibility

## Acceptance Criteria

- Buyer understands seller identity and terms.
- Filters support comparison.
- Transaction steps and platform roles are clear.
