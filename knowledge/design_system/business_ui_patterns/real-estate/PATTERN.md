# Real Estate UI Pattern Pack

Use this pack for property marketplaces, agency sites, listing portals, rental platforms, and agent CRM surfaces.

## Design Posture

- Priority: photos, location, price, filters, contact intent.
- Density: visual for public listings; compact for agent/admin views.
- Motion: `BALANCED_MOTION`.

## Required Views

- Property listing
- Property detail
- Map search
- Agent profile
- Saved/compare

## Required Flows

- `CRUD_FLOW.md` for admin/listing management

## Creative Axes

- Gallery prominence
- Map/list balance
- Contact CTA placement
- Price and feature hierarchy
- Saved/compare behavior

## Acceptance Criteria

- Photos, price, location, and CTA are prominent.
- Filters support serious search.
- Property status and availability are clear.
