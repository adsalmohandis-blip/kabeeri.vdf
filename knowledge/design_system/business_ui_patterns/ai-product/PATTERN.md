# AI Product UI Pattern Pack

Use this pack for chatbots, prompt tools, content generation, document analysis, automation tools, and AI assistants.

## Design Posture

- Priority: input clarity, truthful processing, readable output, iteration.
- Density: balanced with focused input/output regions.
- Motion: `BALANCED_MOTION`.
- Dashboard reference: `ADMIT-ADB01` for governance, usage, and AI status; `ADMIT-ADB05` for clean prompt tools.

## Required Views

- Prompt input
- Suggestions or templates
- Processing or streaming output
- Generated result
- History and versions
- Usage limits and feedback

## Required Flows

- `AI_PRODUCT_FLOW.md`
- `ONBOARDING_FLOW.md`

## Creative Axes

- Prompt surface shape
- Suggestion style
- Streaming indicator treatment
- Output action placement
- Usage and limit visibility

## Acceptance Criteria

- First-run screen explains what the user can ask or upload.
- Processing is visible but not theatrical.
- Output includes copy, regenerate, save, and feedback actions.
- Usage limits and errors are recoverable.
