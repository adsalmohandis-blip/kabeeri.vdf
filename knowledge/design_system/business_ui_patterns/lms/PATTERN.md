# LMS / EdTech UI Pattern Pack

Use this pack for course platforms, lesson systems, quizzes, assignments, progress tracking, and certificates.

## Design Posture

- Priority: learning focus, progress clarity, low distraction.
- Density: balanced; focused inside lessons.
- Motion: `BALANCED_MOTION`.

## Required Views

- Course catalog
- Course detail
- Lesson player
- Quiz/assignment
- Progress/certificate

## Required Flows

- `ONBOARDING_FLOW.md`

## Creative Axes

- Lesson navigation style
- Progress visibility
- Quiz feedback tone
- Course card density
- Certificate reveal style

## Acceptance Criteria

- Learners know current progress and next lesson.
- Lesson screen reduces distraction.
- Quiz feedback is clear and accessible.
