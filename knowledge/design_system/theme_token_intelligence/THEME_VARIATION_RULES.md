# Theme Variation Rules

Use these rules to keep generated UI creative without losing governance.

## Vary

- palette preset
- accent strategy
- density
- radius scale
- surface style
- shadow softness
- navigation rhythm
- chart color strategy
- microcopy tone
- motion level

## Do Not Vary Randomly

- state colors
- focus visibility
- contrast minimums
- destructive action styling
- form error meaning
- RTL direction behavior
- spacing scale semantics

## Prompt Rule

Reference the exported token file path in prompts. Do not paste the full token
object unless the implementation tool cannot read project files.
