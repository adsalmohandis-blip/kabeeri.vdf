# Image And Media Performance Rules

## Images

- Use explicit width, height, or aspect-ratio.
- Use responsive image sources when multiple viewport sizes matter.
- Use modern formats such as WebP or AVIF when the stack supports them.
- Compress images before shipping.
- Lazy-load below-the-fold images.
- Do not lazy-load the first-viewport primary image.
- Avoid dark, blurred, oversized, or decorative images when the user needs to inspect the product or place.

## Media

- Avoid autoplaying video unless it is justified and muted.
- Provide poster images with dimensions for video.
- Do not use video as a substitute for actual product content.
- Defer heavy galleries, maps, embeds, and third-party widgets until needed.

## Commerce And Real Estate

- Product and property images must be crisp and inspectable.
- Gallery thumbnails need stable dimensions.
- Variant image swaps should avoid layout shifts.
- Use progressive placeholders that preserve image ratio.
