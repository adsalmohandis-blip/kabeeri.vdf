# FOUND-UI01 Foundation Sites Foundation

Foundation Sites 6.9.0 is an approved Kabeeri UI foundation for professional responsive websites, mature grid systems, and highly customizable public or app interfaces.

Use it when the project needs a structured responsive framework and the team is comfortable with Sass settings, framework configuration, and selective JavaScript plugin use.

## Best For

- professional public websites;
- enterprise marketing and content sites;
- custom responsive app shells;
- teams that want a mature grid and design-system configuration;
- projects that may also plan responsive email design separately.

## Acceptance Rules

- Foundation Sites version is pinned to `6.9.0` or upgraded with a recorded design-system decision.
- Sass settings map to project design tokens.
- JavaScript plugin usage and initialization policy are documented.
- Foundation is not mixed with another full CSS framework without a migration decision.
- Responsive, accessibility, and RTL behavior are tested for Arabic surfaces.
