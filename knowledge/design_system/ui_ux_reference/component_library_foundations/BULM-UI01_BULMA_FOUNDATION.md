# BULM-UI01 Bulma Foundation

Bulma 1.0.4 is an approved Kabeeri UI foundation for simple responsive interfaces that need a CSS-only framework, readable classes, Flexbox layout, and a lighter alternative to Bootstrap.

Use Bulma when the project needs fast structure without JavaScript plugins or React-only components. Kabeeri still requires approved tokens, page specs, component contracts, responsive behavior, and visual acceptance checks.

## Best For

- simple public websites and internal tools;
- MVPs that need responsive layout without framework lock-in;
- teams that want CSS-only components and helpers;
- Bootstrap alternatives where JavaScript plugins are not required.

## Acceptance Rules

- Bulma version is pinned to `1.0.4` or upgraded with a recorded design-system decision.
- Sass variables or CSS custom properties map to project tokens.
- Allowed components, layout helpers, form rules, and responsive behavior are documented.
- Any JavaScript behavior is owned by project component contracts.
- RTL and accessibility behavior are tested for Arabic surfaces.
