# SHAD-UI01 shadcn/ui Foundation

shadcn/ui with CLI 4.7.0 is an approved Kabeeri UI foundation for React and Next.js projects that use Tailwind and want owned, editable component source instead of a traditional installed component package.

Use shadcn/ui when the product needs modern component composition, Radix-style accessible primitives, strong customization, and local ownership of UI source files.

## Best For

- React and Next.js applications;
- Tailwind-based SaaS dashboards and portals;
- modern forms, dialogs, menus, command palettes, tables, and settings screens;
- teams that want copied component source they can edit and govern.

## Acceptance Rules

- shadcn CLI version is pinned to `4.7.0` or upgraded with a recorded design-system decision.
- Kabeeri records that shadcn/ui copies component source into the app instead of behaving like a traditional npm component dependency.
- Tailwind tokens and CSS variables map to project design tokens.
- The copied components list and ownership rules are documented.
- Accessibility, keyboard behavior, dark mode, and RTL behavior are tested.
