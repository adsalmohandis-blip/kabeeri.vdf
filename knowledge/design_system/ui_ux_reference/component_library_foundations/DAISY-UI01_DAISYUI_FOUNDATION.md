# DAISY-UI01 daisyUI Foundation

daisyUI 5.5.19 is an approved Kabeeri UI foundation for Tailwind projects that need faster component classes, themeable UI, and MVP-friendly building blocks.

Use daisyUI only when Tailwind is already selected or explicitly approved. It adds component classes on top of Tailwind, so its themes and class conventions must be governed like any other design-system layer.

## Best For

- Tailwind MVPs and prototypes;
- SaaS dashboards and public pages that need fast component classes;
- forms, buttons, cards, alerts, menus, modals, and theme switching;
- teams that want Tailwind flexibility with less utility repetition.

## Acceptance Rules

- daisyUI version is pinned to `5.5.19` or upgraded with a recorded design-system decision.
- Tailwind setup is approved first.
- daisyUI themes map to project design tokens.
- Allowed component classes and theme names are documented.
- Dark mode, responsive behavior, accessibility, and RTL behavior are tested.
