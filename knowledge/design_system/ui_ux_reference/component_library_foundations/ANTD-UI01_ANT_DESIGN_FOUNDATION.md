# ANTD-UI01 Ant Design Foundation

Ant Design 6.3.7 is an approved Kabeeri UI foundation for React enterprise applications, admin dashboards, ERP, CRM, and data-heavy backoffices.

Use Ant Design when the product needs a complete operational component system with strong tables, forms, filters, date pickers, modals, and enterprise navigation.

## Best For

- ERP, CRM, BI, HR, inventory, accounting, and helpdesk systems;
- dense admin dashboards and backoffice workflows;
- products with many forms, tables, filters, and bulk actions;
- teams that prefer a complete React component system.

## Acceptance Rules

- Ant Design version is pinned to `6.3.7` or upgraded with a recorded design-system decision.
- Ant Design theme tokens map to project design tokens.
- ConfigProvider, locale, direction, density, and app shell policy are documented.
- Data-table and form contracts exist before implementation.
- Accessibility, responsive tables, density, and RTL behavior are tested.
