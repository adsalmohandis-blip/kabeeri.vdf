# MUI-UI01 Material UI Foundation

Material UI / MUI 9.0.1 is an approved Kabeeri UI foundation for React applications that need themeable Material Design components, forms, navigation, dashboards, and component-heavy workflows.

Use MUI when the project is React-based and the product should follow or adapt Material Design through a governed theme.

## Best For

- React applications and dashboards;
- SaaS admin panels;
- component-heavy forms and settings screens;
- products that benefit from Material Design conventions;
- teams that need a mature, accessible, themeable React component system.

## Acceptance Rules

- MUI version is pinned to `9.0.1` or upgraded with a recorded design-system decision.
- The MUI theme maps to project design tokens.
- Emotion/styling setup and SSR behavior are documented.
- Component variants are approved before page implementation.
- Responsive, accessibility, dark mode, and RTL behavior are tested.
