# 14 - Open Source Strategy

Kabeeri VDF can be useful as an open-source framework when its rules are clear,
its CLI is stable, and its docs explain how AI-assisted development stays
governed.

## Principles

- keep examples safe and free of secrets
- document contribution expectations
- separate product roadmap from client-specific work
- require tests for runtime behavior
- keep release decisions explicit
