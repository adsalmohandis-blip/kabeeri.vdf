# 04 — Product Roadmap

## Phase 1 — GitHub Repository

Publish docs, generators, templates, questionnaires, prompt packs, schemas, and examples.

## Phase 2 — CLI

Add commands such as:

```bash
kvdf new my-project --profile standard --lang en
kvdf validate
kvdf export-ai-handoff
```

## Phase 3 — VS Code Extension

Add questionnaire navigation, task tracking, prompt export, and acceptance checklist views.

## Phase 4 — Desktop App

Create a local GUI for non-technical users.

## Phase 5 — SaaS

Add team workspaces, permissions, template marketplace, AI provider integrations, and review workflows.
