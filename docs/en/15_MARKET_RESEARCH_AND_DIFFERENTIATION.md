# 15 - Market Research and Differentiation

Kabeeri VDF is positioned differently from normal coding frameworks and normal
AI chat usage.

## Differentiation

- It governs the work before code starts.
- It turns natural language into scoped tasks and prompt packs.
- It records AI runs, costs, sessions, acceptance, and handoff state.
- It supports multiple implementation stacks instead of replacing them.
- It keeps owner verification and policy gates explicit.

## Not A Replacement For

- Laravel, React, Expo, Flutter, Django, or .NET
- GitHub
- VS Code
- project management tools
- official security or legal review

Kabeeri sits above those tools and helps them work together.
