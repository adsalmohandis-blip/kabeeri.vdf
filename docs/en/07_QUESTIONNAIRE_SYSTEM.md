# 07 - Questionnaire System

## Goal

The questionnaire system turns a broad idea into precise answers that AI can
use safely.

## Question Rules

1. Questions must be understandable to beginners.
2. Technical questions need short explanations or examples.
3. Vague questions should be replaced with clear choices.
4. Do not ask what the folder is for; explain that in the introduction.
5. Ask only what is needed for the current phase.

## Recommended Shape

```text
1. Folder or area introduction.
2. Why this area matters.
3. What AI may generate after answers are provided.
4. Instructions for the owner.
5. Questions.
6. Answer space.
7. Instructions for AI after the file is answered.
```

## Runtime Use

The CLI supports questionnaire export, status, adaptive flow, coverage, and
task generation from missing or confirmed answers.
