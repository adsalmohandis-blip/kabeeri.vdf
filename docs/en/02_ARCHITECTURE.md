# 02 — Framework Architecture

```text
Kabeeri VDF
├── Project Skeleton Layer
├── Questionnaire Layer
├── Documentation Generation Layer
├── Prompt Pack Layer
├── Task Execution Layer
├── Acceptance & Review Layer
```

The framework starts before code. It creates project structure, asks beginner-friendly questions, generates folder-specific documentation, then produces AI-ready prompts and tracked implementation tasks.
