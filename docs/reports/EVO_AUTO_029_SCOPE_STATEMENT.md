# EVO_AUTO_029 Scope Statement

## Priority

- ID: `evo-auto-029-cli-surface`
- Title: `CLI Capability Surface`
- Source: `new_features_docs_study`

## Scope

This priority maps each capability area to a discoverable CLI command family and docs reference so the operational entry point is obvious before work starts.

The scope includes:

- CLI capability mapping
- docs/runtime/report linkage
- complete surface coverage

## Out of scope

- hiding capability ownership in a generic list
- removing runtime or report references
- treating the CLI surface as a static appendix only

## Success criteria

- all capabilities have a visible CLI family
- docs and runtime references stay aligned
- the surface map remains complete
- downstream documentation can rely on the matrix
