# EVO_AUTO_032 Scope Statement

## Priority

- ID: `evo-auto-032-scale-packs`
- Title: `Scale-Specific Packs`
- Source: `new_features_docs_study`

## Scope

This priority adds packs for very large systems so KVDF can scale without forcing every project through the same context.

The scope includes:

- scale-pack bundle selection
- enterprise/high-risk pack coverage
- project-profile integration

## Out of scope

- using a single generic pack for every project
- hiding bundle selection in chat history
- changing the project profile system itself

## Success criteria

- large systems get broader bundles
- the selected packs are persisted
- the bundle recommendations remain explicit
- downstream prompt composition can reuse the selected pack set
