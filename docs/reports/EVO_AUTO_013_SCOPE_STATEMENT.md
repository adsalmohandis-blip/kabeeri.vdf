# EVO_AUTO_013 Scope Statement

## Priority

- ID: `evo-auto-013`
- Title: `Team GitHub sync deepening`
- Source: `owner_conversation`

## Scope

This priority makes GitHub feedback deeper and more explicit for team mode by tracking issue, PR, status, and comment feedback separately while keeping solo mode local-only.

The scope includes:

- GitHub team feedback status
- GitHub team feedback log
- issue/PR/status/comment record separation
- optional sync summaries without mandatory remote writes

## Out of scope

- making GitHub mandatory for solo work
- changing unrelated planner or evolution behavior
- replacing the local workspace as the source of truth
- collapsing feedback types into a single generic record

## Success criteria

- team mode can summarize GitHub feedback cleanly
- solo mode stays local-only
- the sync layer remains advisory for solo work
- issue/PR/status/comment feedback stays distinct
