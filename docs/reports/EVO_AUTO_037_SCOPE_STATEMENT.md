# EVO_AUTO_037 Scope Statement

## Priority

- ID: `evo-auto-037-source-normalization`
- Title: `Source Folder Normalization`
- Source: `new_features_docs_study`

## Scope

This priority normalizes the imported source folders into safe lowercase structure with preserved mappings.

The scope includes:

- source-package normalize output
- lowercase aliases
- preserved mappings

## Out of scope

- reintroducing the historical source folder
- dropping mapping evidence
- losing traceability between roots, sections, and destinations

## Success criteria

- the normalization map remains readable
- preserved mappings remain intact
- the source package can be understood as historical without losing context
