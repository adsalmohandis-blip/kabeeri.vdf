Viber pipeline (simple):
1) App idea and workspace sync
2) Plan generation for the app track
3) Documentation plan and docs creation
4) Version/evolution/task output
5) Review + execution + feedback
6) Gate evaluation + security checks
7) Handoff and publish readiness
8) Dashboard + next version/action
Questions block:
Q) Ask clarifying questions, capture constraints, and lock the initial scope
Keep the loop simple: plan → build → validate → gate → publish readiness → next step.
