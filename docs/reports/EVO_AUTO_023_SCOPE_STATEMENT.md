# EVO_AUTO_023 Scope Statement

## Priority

- ID: `evo-auto-023-lifecycle-engine`
- Title: `Lifecycle Engine`
- Source: `new_features_docs_study`

## Scope

This priority tracks tasks through intake, readiness, execution, validation, and closure as a visible lifecycle.

The scope includes:

- task lifecycle stages
- blocked and archived visibility
- a direct lifecycle board query
- stage-aware runtime state

## Out of scope

- replacing lifecycle stages with a simple binary status
- removing archive history
- hiding blocked work
- changing unrelated planning behavior

## Success criteria

- the lifecycle board is queryable
- the stage model is visible and distinct
- blocked and archived states are preserved
- downstream task governance can use the lifecycle board as the source of truth
