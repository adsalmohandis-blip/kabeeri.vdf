# EVO_AUTO_034 Scope Statement

## Priority

- ID: `evo-auto-034-developer-onboarding`
- Title: `Developer Onboarding Flow`
- Source: `new_features_docs_study`

## Scope

This priority gives new developers a guided first-session path that explains how to enter, route, and resume work safely.

The scope includes:

- onboarding guidance
- persisted onboarding report
- safe opening sequence

## Out of scope

- replacing the onboarding guide with generic help text
- hiding the route command or blocked features
- making onboarding rely on chat memory

## Success criteria

- the onboarding report is persisted
- the route and resume sequence is explicit
- the guide can be reloaded later
- downstream sessions can start safely from the recorded path
