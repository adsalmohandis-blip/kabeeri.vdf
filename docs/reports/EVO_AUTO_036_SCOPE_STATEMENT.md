# EVO_AUTO_036 Scope Statement

## Priority

- ID: `evo-auto-036-capability-doc-matrix`
- Title: `Capability-to-Documentation Matrix`
- Source: `new_features_docs_study`

## Scope

This priority requires every capability to have docs, CLI, runtime, tests, and report links in one matrix.

The scope includes:

- the capability matrix report
- grouped owner and group coverage
- complete surface coverage

## Out of scope

- flattening the matrix into a narrative summary
- accepting partial coverage
- removing owner/group organization

## Success criteria

- the matrix remains complete
- all required surface links remain visible
- the report can be reloaded later without rebuilding the mapping
