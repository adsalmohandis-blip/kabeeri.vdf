# EVO_AUTO_028 Scope Statement

## Priority

- ID: `evo-auto-028-docs-site-publishing`
- Title: `Docs Site Deep Publishing`
- Source: `new_features_docs_study`

## Scope

This priority publishes the docs site coverage report so generated docs remain traceable to capability, governance, intake, operations, examples, and support coverage.

The scope includes:

- docs site coverage reporting
- family-level publishing completeness
- validation-aligned publishing state

## Out of scope

- replacing the docs site with an ad hoc export
- hiding missing pages or incomplete families
- changing capability ownership inside the docs site

## Success criteria

- all families remain published
- coverage stays complete
- missing pages would be explicit if they appeared
- downstream docs-generation work can rely on the published site
