# EVO_AUTO_031 Scope Statement

## Priority

- ID: `evo-auto-031-project-profiles`
- Title: `Project Profile System`
- Source: `new_features_docs_study`

## Scope

This priority lets the profile shape the depth, speed, and governance footprint of the current project.

The scope includes:

- project profile routing
- profile status/reporting
- scale-pack recommendations
- intake groups and next actions

## Out of scope

- treating the profile as a one-off suggestion only
- hiding the scale-pack recommendation
- changing intake without a persisted profile result

## Success criteria

- the profile is durable
- the delivery mode and scale packs remain visible
- the routing result can be resumed later
- downstream planning can use the profile as source of truth
