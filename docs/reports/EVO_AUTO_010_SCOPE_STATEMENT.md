# EVO_AUTO_010 Scope Statement

## Priority

- ID: `evo-auto-010`
- Title: `Low-cost project start mode`
- Source: `owner_conversation`

## Scope

This priority makes the initial project start path cheaper and shorter by routing to a durable profile and recommending compact prompt packs before broad generation begins.

The scope includes:

- project profile routing
- compact prompt-pack recommendations
- short resume guidance for the next action
- compatibility with questionnaire and blueprint planning

## Out of scope

- redesigning the project-profile engine
- changing unrelated delivery or questionnaire semantics
- replacing prompt-pack composition
- broad task generation that ignores the compact-start constraint

## Success criteria

- the runtime can identify a low-cost profile path
- prompt-pack recommendations stay focused
- the next action remains short and resumable
- the profile record persists for later sessions
