# EVO_AUTO_040 Scope Statement

## Priority

- ID: `evo-auto-040-searchable-reference`
- Title: `Searchable Reference Surface`
- Source: `new_features_docs_study`

## Scope

This priority makes the new capability map searchable by track, capability, command, phase, and report type.

The scope includes:

- searchable capability index
- multi-facet filtering
- registry/surface/matrix/roadmap search space

## Out of scope

- manual-only reference lookup
- losing facet consistency between sessions
- reducing the surface to a non-report help message

## Success criteria

- the search index remains readable
- the same facets stay available later
- the surface remains a durable report
