# EVO_AUTO_035 Scope Statement

## Priority

- ID: `evo-auto-035-governance-expansion`
- Title: `Governance Expansion`
- Source: `new_features_docs_study`

## Scope

This priority expands governance coverage so trust, safety, privacy, compliance, and extensibility stay visible in the workspace governance report.

The scope includes:

- governance report output
- coverage dimensions
- workspace and publish targets
- warning visibility

## Out of scope

- turning governance into a generic help page
- hiding warnings behind a fully green display
- removing publish-target distinctions

## Success criteria

- the governance report remains accessible
- coverage dimensions remain explicit
- the report keeps warning signals visible
- downstream sessions can resume from the recorded governance state
