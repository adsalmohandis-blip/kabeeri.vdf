# 05 — دورة حياة المشروع داخل الفريمورك

## المرحلة 1 — إنشاء Skeleton

المستخدم يختار Lite أو Standard أو Enterprise. يتم إنشاء فولدرات المشروع مع ملف أسئلة داخل كل فولدر.

## المرحلة 2 — إجابة الأسئلة

يبدأ المستخدم بالفولدرات الأساسية:

1. Strategy.
2. Foundation Architecture.
3. Release Specs.
4. Database Architecture.
5. Execution Plan.

## المرحلة 3 — توليد مستندات الفولدرات

كل ملف أسئلة مجاب عنه يتم إرساله إلى AI لتوليد مستندات الفولدر نفسه فقط.

## المرحلة 4 — اختيار إطار الكود

بعد اكتمال المستندات، يتم اختيار Laravel أو .NET أو Next.js أو غيره.

## المرحلة 5 — توليد Prompt Pack

يتم توليد برومبتات مرتبة للتنفيذ.

## المرحلة 6 — التنفيذ

كل Prompt يتم تنفيذه كـ Task منفصلة.

## المرحلة 7 — المراجعة والقبول

لا يتم اعتبار أي جزء مكتملًا إلا بعد Acceptance Checklist.

## المرحلة 8 — الإطلاق

تجهيزات DevOps و QA و Security و Documentation.

## المرحلة 9 — التطوير المستقبلي

أي إضافة مستقبلية تبدأ من Extension Layer.
