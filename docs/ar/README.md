# فهرس الوثائق العربية

هذا الفهرس هو المدخل العربي الرسمي لوثائق Kabeeri VDF. يجب أن يظل متطابقا في البنية مع `docs/en/README.md`: نفس أرقام الملفات، نفس الموضوعات، ونفس ترتيب القراءة.

## قاعدة التوحيد

- الملفات المرقمة من `01` إلى `20` هي الملفات الأساسية المعتمدة.
- لكل ملف عربي ملف إنجليزي بنفس الرقم والاسم داخل `docs/en/`.
- إذا زادت التفاصيل في لغة واحدة، يجب إضافة نفس الفكرة في اللغة الأخرى في أقرب تحديث.
- التفاصيل التشغيلية الحديثة تعيش في `docs/SYSTEM_CAPABILITIES_REFERENCE.md`، أما هذه الملفات فهي شرح تأسيسي ومنهجي.

## ملفات إنجليزية قديمة

توجد في `docs/en/` بعض ملفات إنجليزية قديمة بترقيم سابق. تم الاحتفاظ بها حتى لا تنكسر الروابط القديمة، لكن لا يجب استخدامها كمرجع رئيسي في الروابط الجديدة. استخدم دائما الملفات الأساسية المرقمة أعلاه.

## قاعدة الصيانة

عند إضافة أو تعديل وثيقة أساسية:

1. حدث النسخة العربية والنسخة الإنجليزية معا.
2. حدث فهرس `docs/ar/README.md` و`docs/en/README.md`.
3. حدث `docs/BILINGUAL_DOCUMENTATION_PARITY.md` إذا تغيرت قاعدة التوحيد.
4. حدث `docs/reports/BILINGUAL_DOCS_PARITY_REPORT.md` عند عمل مراجعة توحيد جديدة.
