# 20 — قواعد استخدام AI داخل الفريمورك

## للذكاء الاصطناعي

عند استلام ملف من Kabeeri VDF:

1. اقرأ دور الفولدر أولًا.
2. استخدم إجابات المستخدم كمصدر أساسي.
3. لا تفترض مميزات كبيرة غير مذكورة.
4. لا تكتب كود إلا إذا كان الملف Prompt Pack أو Execution Task.
5. لا تنشئ ملفات فولدرات أخرى.
6. اكتب النواقص والافتراضات بوضوح.
7. استخدم لغة بسيطة.

## للمستخدم

- لا ترسل كل المشروع للـ AI مرة واحدة.
- ابدأ بفولدر واحد.
- راجع الناتج.
- لا تنتقل للكود قبل اكتمال المستندات الأساسية.
- استخدم Task Tracking عند بدء التكويد.

## للمساهمين

أي Template أو Prompt يجب أن يحتوي على حدود واضحة لما لا يجب فعله.
