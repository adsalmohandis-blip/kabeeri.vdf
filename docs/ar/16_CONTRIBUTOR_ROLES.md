# 16 — أدوار المساهمين

## Framework Lead

يحافظ على الرؤية والقرارات الكبرى.

## Documentation Maintainer

يراجع docs ويمنع الغموض.

## Questionnaire Designer

يكتب أسئلة سهلة لكنها قوية.

## Prompt Pack Engineer

يبني برومبتات قابلة للتنفيذ والمراجعة.

## Template Maintainer

يحافظ على قوالب Word/Markdown.

## Tooling Developer

يبني CLI و VS Code Extension وأدوات التحقق.

## QA Reviewer

يراجع Checklists ويختبر أمثلة حقيقية.

## Security Reviewer

يتأكد من أن القوالب لا تشجع على ممارسات غير آمنة.
