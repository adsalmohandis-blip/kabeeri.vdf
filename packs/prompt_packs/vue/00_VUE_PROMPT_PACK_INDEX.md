# 00 — Vue.js Prompt Pack Index

## Goal

Provide a safe, beginner-friendly Vue.js prompt pack for building frontend SPAs and UI layers step by step.

## What this pack is not

This pack is not a Vue installer, Node.js installer, package manager, or full app generator.

## Best use cases

```text
Frontend SPA
Admin dashboard UI
Embedded frontend
Vue UI for Laravel/.NET/FastAPI/NestJS backend
Vue UI for Supabase/Firebase/Strapi/Shopify
```
