# 00 — Angular Prompt Pack Index

## Goal

Provide a safe, beginner-friendly Angular prompt pack for building frontend apps step by step.

## What this pack is not

This pack is not an Angular installer, Angular CLI setup tool, package manager, or full app generator.

## Best use cases

```text
Enterprise frontend
Admin dashboard
Internal system UI
Large SPA
Frontend for .NET/Spring Boot/Laravel/FastAPI/NestJS APIs
```
