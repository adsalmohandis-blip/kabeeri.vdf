# Frontend Prompt Packs Overview

This package adds standalone frontend prompt packs for:

```text
react/
vue/
angular/
```

These are different from meta-framework packs such as:

```text
nextjs/
nuxt-vue/
sveltekit/
astro/
```

## Difference

- React.js, Vue.js, and Angular are frontend application frameworks/libraries.
- Next.js, Nuxt, SvelteKit, and Astro add routing/rendering/build conventions on top of frontend ecosystems.
- Use standalone React/Vue/Angular when the project is a SPA, dashboard, embedded frontend, or frontend connected to a separate backend.
