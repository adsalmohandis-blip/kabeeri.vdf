# Lite Example Product Brief

## Product name

```text
Simple Booking Request App
```

## Product goal

Allow visitors to submit a booking request through a simple form.

## Target users

```text
- Website visitors
- Business owner
```

## Main user action

```text
Visitor fills the form and submits a booking request.
```

## First version features

```text
- Public landing page
- Booking form
- Required fields validation
- Success message after submit
- Basic request storage or email notification
```

## Suggested form fields

```text
- Full name
- Phone number
- Email
- Requested date
- Requested service
- Notes
```

## Success message

```text
Thank you. Your booking request has been received. We will contact you soon.
```

## Not included in V1

```text
- Login
- Payment
- Calendar sync
- Advanced admin panel
- SMS notifications
```
