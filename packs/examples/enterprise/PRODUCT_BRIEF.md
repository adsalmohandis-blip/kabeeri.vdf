# Enterprise Example Product Brief

## Product name

```text
Multi-Branch Service Operations Platform
```

## Product goal

Help a multi-branch organization track service requests, team assignments, status changes, and management reviews.

## Target users

```text
- System owner
- Admin
- Branch manager
- Team member
- Viewer/reviewer
```

## First version features

```text
- Branch management
- User roles
- Service request creation
- Assignment workflow
- Status workflow
- Manager notes
- Audit-friendly history
- Basic dashboard
```

## Main records

```text
Organization
Branch
User
Role
ServiceRequest
Assignment
StatusHistory
ReviewNote
```

## Suggested statuses

```text
new
assigned
in_progress
waiting_for_customer
completed
cancelled
```

## Not included in V1

```text
- Payroll
- Accounting
- Full inventory
- Complex BI
- Advanced automation
- External compliance integrations
```
