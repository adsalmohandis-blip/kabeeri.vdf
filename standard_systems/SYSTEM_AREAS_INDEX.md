# System Areas Index

Kabeeri v5 tracks 53 system areas. Use `kvdf capability list` for the machine-readable runtime list.

Each area can be activated as:

```text
required / optional / deferred / not_applicable / unknown / needs_follow_up
```

The index includes product, users, permissions, journeys, frontend splits, backend, database, authentication, authorization, admin, settings, branding, dashboard customization, notifications, search, files, analytics, audit logs, security, integrations, payments, SEO, localization, accessibility, performance, error handling, testing, deployment, publish state, backup, monitoring, documentation, support, legal, content, workflows, multi-tenancy, feature flags, import/export, automation, AI features, data governance, tenant customization, templates, dynamic forms, versioning, and Kabeeri control.
