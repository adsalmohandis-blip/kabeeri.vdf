# Plugin Scope

In scope:

- all possible app categories as data
- active catalog filters
- profile selection
- source intake routing
- questionnaire packs
- spec and micro-doc contracts
- roadmap tracks and order
- workspace planning
- evidence and approval gates

Out of scope:

- direct app implementation
- hardcoded category branching in business logic
- hidden or silent category activation
- destructive workspace operations
- publishing without approval

The plugin sits between app creation intent and the KVDOS app pipeline.
