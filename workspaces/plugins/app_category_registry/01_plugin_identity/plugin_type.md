# Plugin Type

`app_category_registry` is a `core_governance_plugin` and `workspace_planning_plugin` for KVDOS.

It owns:

- category visibility
- category profile building
- compatibility checks
- source intake routing
- questionnaire routing
- spec resolution
- roadmap ordering
- workspace planning
- evidence writing

It does not own:

- app implementation code
- app package generation
- direct install or marketplace publishing

The package root remains `04_plugin_package/`.
