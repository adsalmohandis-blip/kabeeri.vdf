# Market Position

`app_category_registry` is a platform/governance plugin, not a content plugin.

It should be positioned as:

- the app creation category brain
- a safe selector for ready categories
- a pipeline planner for docs, roadmaps, and workspace shape
- a readiness gate for app creation

It is useful whenever a new app needs:

- category selection
- questionnaire routing
- doc resolution
- roadmap ordering
- workspace planning

It should be hidden from casual users and visible as a guided creation engine in the IDE.
