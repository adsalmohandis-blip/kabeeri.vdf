# Schemas

The package is driven by explicit schema contracts.

## Primary schemas

- `category.schema.json`
- `category-group.schema.json`
- `category-readiness.schema.json`
- `category-visibility.schema.json`
- `app-category-profile.schema.json`

## What they protect

- category shape and metadata
- readiness and lifecycle state
- visibility behavior
- selected app profile validity
- deterministic profile planning
