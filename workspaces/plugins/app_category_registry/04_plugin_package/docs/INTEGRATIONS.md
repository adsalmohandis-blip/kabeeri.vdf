# Integrations

`app_category_registry` is the upstream planning layer for new app creation in KVDOS.

It integrates with:

- `plugin_folder_structure`
- `plugin_dev`
- the KVDOS CLI
- `.kabeeri` runtime planning state

It emits category profiles, routing plans, doc contracts, roadmap orders, and workspace plans that downstream tools can consume without reinterpreting the category decision.
