# Plugin Market Position

`app_folder_structure` is a governance plugin, not an app framework.

Its value is to produce consistent, auditable, category-governed app workspaces that another team can continue safely.

Promotion target:

- Owner Track review first
- direct install or marketplace request only after validation and evidence

The plugin is not intended to create user-facing app features directly. It creates the contract surface that other app plugins and teams rely on.
