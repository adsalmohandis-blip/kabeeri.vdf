# Plugin Scope

In scope:

- create governed app workspaces
- validate the canonical app pipeline
- repair missing folders additively
- generate manifests and evidence
- integrate approved category profiles
- keep `08_source/` neutral and stack-adaptive

Out of scope:

- inventing app source trees
- publishing to a marketplace
- direct install promotion without Owner Track
- deleting or renaming user files
- creating plugins from Viber/App Track
