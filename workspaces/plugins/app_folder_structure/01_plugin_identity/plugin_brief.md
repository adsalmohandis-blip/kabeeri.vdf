# Plugin Brief

`app_folder_structure` is the official app-workspace contract engine for KVDF/KVDOS.

It governs how new app workspaces are created, validated, repaired, documented, and audited under:

- `workspaces/apps/<app_slug>/`

It does not invent source layouts. It keeps `08_source/` stack-adaptive and category-governed roadmap internals explicit and safe.
