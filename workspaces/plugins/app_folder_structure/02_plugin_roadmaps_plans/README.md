# Roadmap Plans

This folder holds the implementation planning artifacts for `app_folder_structure`.

It covers:

- architecture planning
- command surface planning
- runtime-state planning
- schema planning
- dashboard planning
- integration planning
- testing planning
- release planning
- marketplace planning

The canonical package lives in `04_plugin_package/`; this folder is the planning layer that describes how the package should behave and what app workspaces it governs.
