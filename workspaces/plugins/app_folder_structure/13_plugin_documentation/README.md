# Documentation

This folder stores the final user-facing documentation for the app workspace contract.

It should explain the same canonical behavior that the package implements:

- app workspaces live under `workspaces/apps/<app_slug>/`
- the top-level pipeline is fixed
- roadmap internals are category-governed
- `08_source/` is stack-adaptive
- manifests, validation, repair, evidence, and approvals are all trackable
