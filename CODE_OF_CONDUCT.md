# Code of Conduct

Kabeeri Vibe Developer Framework is built for learning, collaboration, and practical software creation with AI.

Participants are expected to:

- Be respectful.
- Help beginners.
- Critique ideas, not people.
- Avoid gatekeeping technical knowledge.
- Keep discussions focused on improving the framework.
- Avoid unsafe, illegal, or harmful software guidance.

Maintainers may remove content or block contributors who repeatedly violate these principles.
