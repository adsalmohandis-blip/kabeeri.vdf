## Scope review

```text
- [ ] Output matches the original task.
- [ ] Output does not add future features.
- [ ] Output does not change unrelated folders.
- [ ] Output follows the allowed files list.
- [ ] Output avoids forbidden files.
```

## File review

```text
- [ ] Files are in the correct folder.
- [ ] File names are clear.
- [ ] No duplicate files were created accidentally.
- [ ] No temporary/generated junk files were added.
- [ ] Manifest/schema files match the folder contents if present.
```

## Content review

```text
- [ ] Content is beginner-friendly.
- [ ] Content is structured and easy to scan.
- [ ] Instructions are direct and practical.
- [ ] Technical terms are explained when needed.
- [ ] AI instructions are clear and scoped.
```

## Safety review

```text
- [ ] No real secrets were added.
- [ ] No private keys were added.
- [ ] No production credentials were added.
- [ ] No private user data was added.
- [ ] No unsafe or unrelated instructions were added.
```

## Testing/check review

```text
- [ ] Required checks/tests were run.
- [ ] Manual review was completed.
- [ ] Known failures are documented.
- [ ] Remaining risks are documented.
```
